
TERMS:
- Este é um site estático pronto para GitHub Pages.
- Substitua texto, IPs, imagens e links pelos seus dados reais.

COMO PUBLICAR (rápido):
1) No GitHub, crie o repositório com nome: SEU_USUARIO.github.io (ex: filypesouza.github.io)
2) Envie todos os arquivos (index.html, rules.html, staff.html, downloads.html, whitelist.html e a pasta assets)
3) Vá em Settings → Pages e confirme que o site está publicado (ou use GitHub Actions como já configurado)
4) Acesse: https://SEU_USUARIO.github.io

NOTAS SOBRE A WHITELIST:
- A página de whitelist abre um link para criar uma nova issue no seu repositório (método simples). 
  Substitua 'https://github.com/SEU_USUARIO/SEU_REPOSITORIO/issues/new?title=...' no JavaScript pelo seu repositório
  (ex: https://github.com/filypesouza/filypesouza.github.io/issues/new?title=Whitelist%20-%20NICK&body=...)
- Alternativa: crie um Google Form e cole o link na página de Downloads/Whitelist para receber inscrições.

IMAGENS E VÍDEOS:
- As imagens e o vídeo são links públicos; se preferir, troque por imagens próprias hospedadas em assets/.
